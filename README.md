# GetBingWallpaper
Little **shell** script to get Bing Wallpapers using the RestAPI of [TimothyYe](https://github.com/TimothyYe/bing-wallpaper/blob/master/README.md)
## How to use?
```bash
sh get_bing_wallpaper.sh
```
